Как это работает

1

Авторизуйтесь через Яндекс

Войдите в приложение через OAuth 2.0 — ваш Яндекс ID будет использован для привязки ключа

2

Получите ключ WAWE

Бесплатно через Telegram (до 24ч ожидания) или моментально за 100 ₽

3

Активируйте в приложении

Введите ключ в окне активации. Ключ привяжется к вашему HWID и Яндекс ID с помощью RSA+AES-GCM шифрования

4

Слушайте музыку

Импортируйте треки из Яндекс Музыки, создавайте плейлисты, включайте «Мою волну»

Импорт из Яндекс Музыки

WAWE поддерживает импорт вашей музыки из Яндекс Музыки тремя способами:

  • По ссылке на плейлист — вставьте URL плейлиста Яндекс Музыки (новый UUID-формат или старый)
  • По ссылке на альбом — загрузите все треки альбома
  • Избранное (лайки) — одним нажатием импортируйте все треки, которые вы отметили «Мне нравится»

После импорта WAWE автоматически находит полные MP3-версии треков через HitMotop, так что вы можете слушать их без ограничений прямо в приложении.

Технические детали

Клиент: React 19 + TypeScript + Vite + Tailwind CSS

Сервер активации: Node.js + Express (localhost:3001)

Хранилище: GitHub API (репозиторий Doinwor/WAWE)

Поиск музыки: HitMotop (полные MP3), iTunes (превью), Deezer (превью), Last.fm (информация об артистах)

Шифрование: RSA-SHA256 подпись + AES-256-GCM (PBKDF2) для защиты лицензии

Аудиоплеер: Howler.js
