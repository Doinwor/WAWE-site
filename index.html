Как начать

4 шага до твоей музыки

Получить доступ к WAWE очень просто.

1

Скачай WAWE

Загрузи единственный .exe-файл (88 МБ). Никакой установки — просто запусти и пользуйся.

2

Авторизуйся через Яндекс

Войди в приложение через Яндекс OAuth. Твой Яндекс ID будет использован для активации.

3

Получи ключ WAWE

Напиши на почту для бесплатного ключа (время выдачи не ограничено) или купи моментально — до полудня, обычно в течение часа. Ключ привязывается к твоему аккаунту.

4

Слушай

Импортируй треки из Яндекс Музыки, создавай плейлисты, включай «Мою волну» и наслаждайся.

Импорт

Импорт из Яндекс Музыки

WAWE поддерживает три способа импорта твоей музыки.

🔗 По ссылке на плейлист

Вставь URL плейлиста Яндекс Музыки — новый UUID-формат или старый. Все треки будут импортированы.

💿 По ссылке на альбом

Загрузи все треки альбома целиком. Идеально для прослушивания новых релизов.

❤️ Избранное (лайки)

Одним нажатием импортируй все треки, которые ты отметил «Мне нравится» в Яндекс Музыке.

🎯 Автоподбор аудио

После импорта WAWE сам находит полные MP3-версии через HitMotop. Треки готовы к прослушиванию сразу.

FAQ

Часто задаваемые вопросы

Ответы на самые популярные вопросы.

Что такое WAWE?

+

WAWE — десктопное приложение для Windows, которое позволяет слушать музыку из Яндекс Музыки без ограничений. Оно ищет полные MP3-версии треков, создаёт персональное радио «Моя волна» и даёт полный контроль над твоей музыкальной библиотекой.

Нужна ли подписка на Яндекс Музыку?

+

Нет. WAWE использует твой бесплатный Яндекс ID для импорта треков (названия, обложки, плейлисты). Сами аудиофайлы находятся через открытые источники (HitMotop).

Как получить ключ?

+

Напиши на olehmudruk@yandex.by. Бесплатный ключ выдаётся вручную — время ожидания не ограничено. Платный ключ — до полудня, обычно в течение часа. Один ключ = один Яндекс ID, без ограничения по устройствам.

Это безопасно?

+

Да. WAWE — клиентское приложение, весь код работает локально на твоём компьютере. Токен Яндекс хранится только у тебя. Активация использует RSA-SHA256 подпись и AES-256-GCM шифрование. Никакие данные не передаются третьим лицам.

Почему .exe такой большой?

+

WAWE — это standalone-приложение, собранное через Node.js Single Executable Application. Внутри .exe находится весь Node.js runtime, React-клиент и встроенный сервер. Ничего не нужно устанавливать — просто запусти файл.

Будет ли версия для macOS/Linux?

+

Пока нет. WAWE заточен под Windows. macOS и Linux версии в планах, но без конкретных сроков.
