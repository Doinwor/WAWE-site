Частые вопросы

Что такое WAWE?
WAWE — это отдельный проект, музыкальный плеер. Он агрегирует треки из открытых источников (HitMotop, iTunes, Deezer, SoundCloud, Last.fm) и предоставляет удобный интерфейс для поиска и прослушивания — бесплатно и без подписок.
Это бесплатно?
Да, WAWE полностью бесплатен. Никаких подписок, скрытых платежей или ограничений по времени.
На каких ОС работает?
Пока только Windows. Версии для других операционных систем в разработке — следите за обновлениями.
Нужна ли установка?
Нет. Просто скачайте WAWE.exe и запустите — это портативная программа, которая не требует установки.
Как это работает?
После запуска WAWE.exe поднимает локальный сервер на http://localhost:3001 и автоматически открывает браузер. Через веб-интерфейс вы можете искать треки, создавать плейлисты и слушать музыку.
Работает ли с VPN?
Рекомендуем использовать WAWE без VPN — при активном VPN некоторые треки могут не грузиться или работа может быть ограничена.
Не запускается WAWE.exe?
Нажмите правой кнопкой на WAWE-v1.1.0.d.exe → Свойства → если есть флажок «Разблокировать» — поставьте его → Применить → OK. Windows Defender может блокировать неподписанные приложения.
